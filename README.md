# stunning-sniffle

## To run this project

Set environment variable

> export GITHUB_API_TOKEN=XXXX
> nodemon server.js

## Technologies used

Node - JS runtime
Express - server
Nodemon - devserver(watch server)
Prettier - format code
Eslint - look for errors

Mocha - to run tests
Chai - to write tests
Chai-http - server testing
